# Portfolio — Amine Atamna · BUT3 GEII ESE

Portfolio personnel d'Amine Atamna, étudiant en 3ᵉ année de BUT Génie Électrique et
Informatique Industrielle, parcours **Électronique et Systèmes Embarqués** (IUT de Troyes).

Thème **« Drone Tech & PCB Design »** — dark mode carbone, accents cyan HUD, motifs de
circuits imprimés animés. Site statique (HTML / CSS / JS vanilla), responsive et léger.

## Structure

| Page | Contenu |
|------|---------|
| `index.html` | Profil / présentation (landing) |
| `but1.html` | Première année — fondamentaux |
| `but2.html` | Deuxième année ESE |
| `but3-ese.html` | Troisième année ESE (collection + référentiel AC) |
| `projet-kart.html` | SAÉ Kart électrique (S6) |
| `projet-stage.html` | Stage R&D drone autonome — CapSec / ARODE |
| `projets.html` | Index de tous les projets |
| `contact.html` | Contact |

- `assets/css/style.css` — design system
- `assets/js/app.js` — animations (reveal, typewriter, compteurs, lightbox, traces PCB)
- `medias/` — images, vidéos et documents
